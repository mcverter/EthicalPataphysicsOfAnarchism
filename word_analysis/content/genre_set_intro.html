{% trans "A simple Genre Set" %}

{% trans "This project will sort each of the terms in Levinas's texts into a collection of Genre Sets that correspond to the various thematics that he subjects to analysis. In its most general form, a Genre Set (GS) can be defined as a mapping of a single term to collection of terms. There are several rules to assembling this collection." %}

{% trans "For example, let's start with a term that appears frequently in" %} {% trans "Totality and Infinity" %}: {% trans "Height" %}

{% trans "For now we don't have any members for the Genre Set of HEIGHT except for itself so we can represent it as follows:" %}:

  • HAUTEUR = { hauteur}

{% trans "Adding synonyms to Genre Sets" %}

{% trans "We can add other forms of this word to the GS such as “hauteur” as well as “haut”. We can add to this set words that are synonymous with HEIGHT, such as “elevation” and phrases such as 'the most high'. We can also include other terms that relate somehow to the effectuation of height, such as standing, 'se tenir'" %}.

  • HAUTEUR = { hauteur, haut, elevation, tenir}

{% trans "Adding morphemically similar terms to Genre Sets" %}

{% trans "We are can also include into this set words that are related to HEIGHT through common etymological roots, or which use the the theme of height in their suffix, prefix or elsewhere in their morphology. For example, we can add words that include the morpheme “sur” in this set such as “surgir”, “surplus” and “surmonte” -- as well as “surface”, “survivant”, and “resurrection”" %}

  • HAUTEUR = { hauteur, elevation, tenir, surgir, surplus, surmonte, surface, survivant, resurrection, }

{% trans "Combinations of Genre Sets: Intersections " %}

{% trans "Each of the above terms signifies an intersection that effectuates a combination between HEIGHT and other GSes" %}

  • HAUTEUR = { hauteur, elevation, tenir,', surgir, surplus, surmonte, surface, survivant, resurrection}
  • MATHEMATIQUES = { plus, surplus, positive}
  • GENÈSES = { vivre, vivant, survivant, naissance}
  • PRESENTATION = { face, surface, facade}

{% trans "Homologous Genre Sets and Parallel Transport" %}

{% trans "How do these intersected sets relate to each other? In many ways. Such sets are often homologous to each other. Thus they often appear in constellated in similar sentences to each other -- for example a sentence about the “positivity” can also be understood as a sentence about the 'survivor,' the living being who manifests in a dimension of height." %}

{% trans "These homologous sets transport metaphors in directions parallel to each other. By parallel transport, we mean to indicate that recurrent thematics such as “height” are always paired with polar opposites such as depth. We can thus pair these with expand our sets to include negative values" %}

,
  • HAUTEUR = { hauteur, elevation, grande, tenir,', surgir, surplus, surmonte, surface, survivant, resurrection,}
  • PROFOND = {profond, profondeur, 'approfondissement, ' bas, subir, subjective, subjectivite, subordonne, substance, substitution, }

{% trans "We might also want to expand other sets to include polar opposite values" %}

  • MATHEMATIQUES = { plus, surplus, positive, negative,'}
  • GENÈSES = { vivre, vivant, survivant, naissance, mort mortal, meurtre, tuer, }
  • PRESENTATION = { face, surface, facade, visage}

{% trans "The MATHÉMATIQUES Genre Set" %}

{% trans "In fact, we would argue that the we might consider creating a new set to connote the union of the polar opposites, HEIGHT and DEPTH. Let's call this provisional GS “VERTICAL”. But then we notice right away that the “vertical” can be opposed to the “horizontal”, a word that connotes the phenomenological concept of the horizon as the plane upon which a something given as a manifestation can be understood. Furthermore, all themes can be encompassed in the another provisional GS, DIMENSION. To this GS we can also add the Cartesian dimension of measure, extension." %}

{% trans "What would the word “dimension” have meant for Levinas? Two of Levinas's primary intellectual influences, Henri Bergson and Edmund Husserl, were both mathematicians and philosophers, whose early thinking had been shaped by considerations of the idea of a manifold taken from the non-Euclidean analytic geometry of Bernhard Reimann. Additionally, Levinas would most likely have read books such as Leon Brunschvicg's 1912 book Les etapes de la philosophie mathematique that whose final chapters are “Le probleme des dimensions”and “La position du probleme non euclidien”. " %}

{% trans "Levinas's knowledge of non-euclidean geometry is exposed most prominently in his usage of the word “curvature”, a term whose provenance is the non-Euclidean geometry of Reimann, Gauss, and others; and by extension in the theories of spacetime articulated by Albert Einstein's theories of general and special relativity. " %}
