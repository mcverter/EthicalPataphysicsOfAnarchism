{% trans "A simple Genre Set" %}

{% trans "This project will sort each of the terms in Levinas's texts into a collection of Genre Sets that correspond to the various thematics that he subjects to analysis. In its most general form, a Genre Set (GS) can be defined as a mapping of a single term to collection of terms. There are several rules to assembling this collection." %}

{% trans "For example, let's start with a term that appears frequently in" %} {% trans "Totality and Infinity" %}: {% trans "Height" %}

{% trans "For now we don't have any members for the Genre Set of HEIGHT except for itself so we can represent it as follows:" %}:

  • HAUTEUR = { hauteur}

{% trans "Adding synonyms to Genre Sets" %}

{% trans "We can add other forms of this word to the GS such as “hauteur” as well as “haut”. We can add to this set words that are synonymous with HEIGHT, such as “elevation” and phrases such as 'the most high'. We can also include other terms that relate somehow to the effectuation of height, such as standing, 'se tenir'" %}.

  • HAUTEUR = { hauteur, haut, elevation, tenir}

{% trans "Adding morphemically similar terms to Genre Sets" %}

{% trans "We are can also include into this set words that are related to HEIGHT through common etymological roots, or which use the the theme of height in their suffix, prefix or elsewhere in their morphology. For example, we can add words that include the morpheme “sur” in this set such as “surgir”, “surplus” and “surmonte” -- as well as “surface”, “survivant”, and “resurrection”" %}

  • HAUTEUR = { hauteur, elevation, tenir, surgir, surplus, surmonte, surface, survivant, resurrection, }

{% trans "Combinations of Genre Sets: Intersections " %}

{% trans "Each of the above terms signifies an intersection that effectuates a combination between HEIGHT and other GSes" %}

  • HAUTEUR = { hauteur, elevation, tenir,', surgir, surplus, surmonte, surface, survivant, resurrection}
  • MATHEMATIQUES = { plus, surplus, positive}
  • GENÈSES = { vivre, vivant, survivant, naissance}
  • PRESENTATION = { face, surface, facade}

{% trans "Homologous Genre Sets and Parallel Transport" %}

{% trans "How do these intersected sets relate to each other? In many ways. Such sets are often homologous to each other. Thus they often appear in constellated in similar sentences to each other -- for example a sentence about the “positivity” can also be understood as a sentence about the 'survivor,' the living being who manifests in a dimension of height." %}

{% trans "These homologous sets transport metaphors in directions parallel to each other. Clockwise Rotations of Metaphors

{% trans "The rotations discussed above are full 180 degree rotations from each other, but we would argue that there are other rotational symmetries within the text of Totality and Infinity. Not only are these books divided into sections, but each of these sections are divided into subsections" %}

{% trans "We would argue that it is best to understand these sections as the angular rotations of a clock. This clock represents many things. First of all is a series of shifting perspectives on and adumbrations of a singular space-time moment, what Levinas refers to as 'the chronological order that precedes the logical order. Second, it can be understood as marking out different hours of the day, with the first section on 'interiority' discussing the morning and the second section on 'exteriority' as discussing the evening. ." %}

{% trans "Understanding the most important clock requires us first to investigate the not just the structure but the story being narrated by Totality and Infinity " %}

Totality and Infinity as the Book of Genesis (בְּרֵאשִׁ֖ית)

{% trans "When one weaves together the metaphorical motifs in the Totality and Infinity, one can the narrative of the book as a whole. It is a story of continuous creation, of birth, death, and rebirth. The divisions of Totality and Infinite narrate a story of Genesis, of the generation of gendered generations, a progression of birth, maturation, adulthood, old age, death, and rebirth. Although this storyline progresses sequentially, each step resonates with others such that Levinas seems to be describing the same situation from the perspective of a different character in this family drama. Otherwise than Being presents this same narrative from yet other perspectives, exhibiting dynamics that were latent or implicit in the earlier text." %}
