How to type Accents or Special Characters without changing keyboard language
ASCII Table, ASCII Codes, ALT Codes

    Instructions:

    1. On your keyboard (make sure numlock is on)
    2. Press and hold down the Alt key then type the number code.
    3. Example:  For the letter é : Hold down the Alt key then type 130
    4. Let go of both fingers at the same time.
